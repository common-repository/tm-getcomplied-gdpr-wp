== Installation ==

1. Upload `tm-getcomplied-wp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Options' -> 'GetComplied' and follow the instructions

== Frequently Asked Questions ==

= What is GetComplied? =

GetComplied is the most powerful and flexible software for you to comply with the GDPR EU rules. Whenever you create a website, an app, a portal, a landing page, or a software, the GetComplied mission is to provide the most efficient solution to help you comply with the law.

= How to use? =

1. Create and account at https://www.getcomplied.com?gc_aid=70 (please use the url provided to support the development of this plugin)
2. Get your project id and insert in this plugin.
3. That's it you are good to go!
